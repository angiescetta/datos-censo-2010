# datos-censo-2010
Datos del Censo 2010 en AMBA

Fuente de datos: REDATAM (https://redatam.indec.gob.ar/argbin/RpWebEngine.exe/PortalAction?&MODE=MAIN&BASE=CPV2010B&MAIN=WebServerMain.inl)
